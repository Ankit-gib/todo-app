# 📝 To Do App

A simple To-Do web application built using Flask and SQLite.

![Python](https://img.shields.io/badge/Python-3.x-blue?logo=python)
![Flask](https://img.shields.io/badge/Flask-3.x-lightgrey?logo=flask)
![SQLite](https://img.shields.io/badge/Database-SQLite-green?logo=sqlite)

## Features

- Add new tasks
- Mark tasks as completed / undo
- Edit existing tasks
- Delete tasks
- Tasks are stored in SQLite database
- Clean glassmorphism UI with dark theme

## Installation

1. Clone the repository
```bash
git clone https://github.com/Ankit-gib/todo-app.git
cd todo-app
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Run the application
```bash
python app.py
```

4. Open in browser
```
http://127.0.0.1:5000
```
